# Dataprep

## Dependencies
- cargo
- rustup

## Build
```bash
cargo build
```

## Format

Make sure to run `cargo fmt` before committing.

```bash
cargo fmt # format code
cargo clippy # lint code
```

## Test

```bash
cargo test
```

## Run
```bash
cargo run -- input_dirs <list of input directories> --output_dir <output directory> 
```
